О системе TerraNova

Добро пожаловать в информационную систему агентства недвижимости «TerraNova». Наша платформа предназначена для эффективного управления объектами недвижимости, сделками и клиентской базой.

Основные возможности системы:

  • Управление списком объектов недвижимости с возможностью добавления, редактирования и удаления.
  • Ведение и контроль сделок с недвижимостью.
  • Работа с базой клиентов и сотрудников агентства.
  • Безопасная авторизация и управление правами доступа.

Технологии и архитектура:

Система построена с использованием современных технологий:

  • Backend на Spring Boot с использованием JPA и Hibernate для работы с базой данных.
  • Frontend с использованием Thymeleaf и Bootstrap 5 для удобного и адаптивного интерфейса.
  • Безопасность реализована через Spring Security с поддержкой регистрации и аутентификации пользователей.
